
##🏪 couponmoa-ai

##📌 개요

이 서버는 Couponmoa 프로젝트의 AI 기반 쿠폰 추천 기능을 전담하는 서비스입니다. Google Cloud의 Vertex AI Gemini 모델을 활용해 사용자 키워드와 쿠폰 데이터를 분석하여 최적의 쿠폰을 추천합니다.

##🧩 주요 기능

키워드 기반 쿠폰 추천 (최대 5개 쿠폰 ID 반환)
Vertex AI Gemini와 통합된 AI 프롬프트 처리
JSON 응답 형식으로 추천 결과 제공
쿠폰 서버와의 API 연계 (/api/v1/coupon/recommend-ai 엔드포인트)
